variables

local variable and global variable

  • local variable can be only used inside of a variable
  • global variable can be used in a whole program

way of storing variables

  • static way of storing: the way the program allocates fixed position of storage. (Store variables that exists all the time, e.g: global variable)
  • dynamic way of storing: the way the program allocates storage dynamically. (store variables that may be set or released as the program is executed, including actual parameter, automatic variables)

4 types of storage

  • 1.auto
  • 2.static
  • 3.register
  • 4.extern: variable that is clarified to be used externally
