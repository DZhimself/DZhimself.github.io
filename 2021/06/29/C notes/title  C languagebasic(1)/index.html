Posted on 2021-06-29 In C language notes

hello world

  • a C-language program is made up from a number of header files and functions
  • #include<stdio.h> is a pre-processing instruction. It’s function is to process some work before compilation.
  • function is a unit to achieve the logic behind the code.

standarized programming

  • 1.one instruction or sentence takes up one row
  • 2.sentence inside a function needs indentation. Usually a tab is for an indentation
  • 3.Brackets have to exist in pairs.
  • 4.Semicolon should exist at the end of each excutable sentence.

data types

Data types in C language can be classified as: basic data type, structured data type, pointer data type, void type

basic data types
Basic data type Byte Application Example

Int 2 used to store integer int age=18
Float 4 used to store decimal float price=11.1
Double 8 used to store decimal with more digits double pi=3.1415926

unchangeable constant

  • Constant can be divided as direct constant and notation constant
  • direct constant is the value that can be used directly without needing to be declared
  • notation constant needs to be defined before used

#define POCKETMONEY10

  • Notation constant cannot be changed after been defined

automatic data type transfer

  • fewer bytes can transfer to more bytes
  • More bytes can not be changed to fewer bytes
