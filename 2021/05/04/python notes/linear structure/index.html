the python notes(6):data structure

Posted on 2021-05-04 In python notes

Data structure is to design the way which the data is stored in the computer

  • linear structure: the elements in the structure has a one-on-one relationship
  • tree structure: one-on-more relationship
  • pic structure: more-on-more relationship

linear structure

stack

  • stack is a set of data which can only insert or delete at one end

  • Characteristic: last-in, first-out LIFO

  • top,bottom

  • push insert li.append

  • pop delete. li.pop

  • Get top li.[-1]

queue
  • queue is a set of data which allows data to be inserted and another end to be deleted

  • front goes one point forward. front=(front+1)%MaxSize
  • rear goes one point forward rear=(rear+1)%MaxSize
  • the condition when the queue is empty rear==front
  • the condition when the queue is full (rear+1)%MaxSize==front
Linked list

Linked-list is the set consisted of a series of Nodes. Each node consists of two parts, the domain of data ‘item’ and the pointer’next’. The linked-list is made by the connection between nodes.
