Posted on 2021-02-25 Edited on 2021-03-01 In python notes

列表查找

  • 查找指在一些数据元素中通过一定的方法找出与给定关键字相同的数据元素的过程

  • 列表查找：从列表中查找制定元素

    输入：列表，带查找元素

    输出：元素下标（未找到元素时一般返回none或—1

顺序查找（linear search）
  • 从列表第一个元素开始，顺序进行搜索，直到搜索到列表最后一个元素为止
二分查找（列表必须是有序列表）
  • 大小排序，从中间算取¸         

列表排序

  • 排序是将一组无序的记录序列调整为有序的记录序列

  • 列表排序：将无序列表变成有序列表

  • 常见排序算法

    排序lowB三人组 排序nb三人组 其他排序
    冒泡排序 快速排序 希尔排序
    选择排序 堆排序 计数排序
    插入排序 归并排序 基数排序
冒泡排序
  • 列表每两个相邻的数，如果前面比后面大，则交换这两个数
  • 一趟排序完成后，则无序区减少一个数，有序区增加一个数
